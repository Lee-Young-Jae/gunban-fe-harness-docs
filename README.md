# Gunban FE Harness — Structure Docs

Gunban 프론트엔드 프로젝트의 멀티 에이전트 하네스 구조를 시각화한 정적 문서.

## 라이브 문서

→ **https://lee-young-jae.github.io/gunban-fe-harness-docs/**

## 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 자체 완결 HTML (Mermaid CDN만 외부 의존) |

## 정본

이 repo는 **deploy 미러**다. 정본 HTML은 Gunban 프로젝트의 다음 경로에 있다:

```
Gunban/.claude/docs/harness-structure.html
```

업데이트 시 Gunban 측의 `scripts/sync-docs.sh`를 실행해 본 repo로 복사 후 push.
